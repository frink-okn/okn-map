OKN Map

This is a prototype of the Proto-OKN Map.

It is generated through the following series of steps:

  • RDF data is processed into a LinkML schema using a set of scripts.
  • This schema is then itself turned into RDF using the LinkML runtime's RDF generator.
  • The RDF representation of that schema is then directly queried with SPARQL using this interface.

How is this used?

  • At the start only Theme 1 graphs (in green) should be visible.
  • Clicking on one of the Theme 1 graphs will display arrows pointing to other Theme 1 graphs, as well as other RDF ontologies (in gray), that are referred to by that graph.
  • Clicking on a graph with references to or from it will expand to show a set of classes defined in that graph.
  • Clicking on a class will, if there are mappings of that class to other classes, show links to those other classes.
